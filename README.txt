To run the project follow the below steps:-

1. Install the node modules using the command

npm install

2. Now, run the project using the command

npm start

3. You dont need to worry about DB as it is already hosted in the cloud. Now checking test if your API is running on 

http://localhost:5000/api/v1

which will give you "API running" response. If you are unable to access the API on port 5000 then

in .env file add PORT=YOURCUSTOMPORTNUMBER and then access it on this port number you provided or the by default port number is 6000 for the app.

4. For more details on API, please check APIDetails.txt
